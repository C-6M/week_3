# 选做题2：给 Agent Loop 加「记忆」功能 — 思路
### 1. 加一个存记忆的地方
可以在 run_agent() 里用一个列表来存历史对话。每轮对话存用户说的和Agent回复的。
如果想让记忆在关程序后还在，可以写到 JSON 文件里。这样下次打开时从文件加载回来。
### 2. 在循环里每轮自动记录
在目前的 while True 循环里，每次用户输入内容后，把这句话加到记忆列表里。等工具执行完了，把返回的结果也加到记忆列表里。
### 3. 增加查询记忆的命令
用户说"之前"、"上次"这些词时，去记忆列表里搜关键词，把匹配的历史记录展示出来。
然后可以在 TOOLS 字典里多注册一个工具，专门负责翻记忆。decide_tool 函数里加对应的关键词判断。例如查天气时，如果短期记忆里 1 小时内已经查过这个城市，可以提示用户，避免重复请求。
